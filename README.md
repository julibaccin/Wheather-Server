Console: 

git clone https://github.com/julibaccin/Wheather-Server.git

cd wheather-server

## 1 Install Package

npm install

## 2 Set .env

Create a .env file, and set:

PORT=?

IP_API_URL=http://ip-api.com/json/

OPEN_WEATHER_URL=http://api.openweathermap.org/data/2.5/

OPEN_WEATHER_KEY=?

## 3 Run Tests

npm run test

## 4 Run Server

npm start
